# 🚀 Ultimate Space Invaders

A feature-packed Space Invaders clone with boss battles, power-ups, save system, and stunning visual effects! Built with Python and Pygame.

## ✨ Features

### 🎮 Core Gameplay
- Classic Space Invaders mechanics with modern enhancements
- 3 enemy types with different health and point values
- Increasing difficulty with each level
- Smooth controls and responsive shooting

### 🐉 Boss Battles
- Epic boss fights every 5 levels
- 4 unique attack patterns that rotate
- Boss health bar and pattern indicator
- Massive explosion effects

### ⚡ 7 Power-Ups
| Power-Up | Effect | Duration |
|----------|--------|----------|
| ⚡ Rapid Fire | Faster shooting | 10 seconds |
| 🛡️ Shield | Invincibility | 10 seconds |
| 💥 Multi-Shot | Triple lasers | 10 seconds |
| 2x Points | Double score | 10 seconds |
| 🐢 Slow Time | Slows enemies | 10 seconds |
| ❤️ Heal | Restores 1 life | Instant |
| 💣 Nuke | Destroys all enemies | Instant |

### 🎨 Visual Features
- 3 dynamic backgrounds (Space, Nebula, Space Warp)
- Twinkling star effects
- Particle explosions
- Power-up collection animations
- Boss battle special effects

### 💾 Save & Load System
- Save your progress with Ctrl+S
- Load saved games with L key
- Persists score, level, and lives

### 🏆 High Score System
- Top 10 scores saved permanently
- Name entry for high scores
- Rank display after game over

### 🔊 Audio Features
- Laser, explosion, power-up sounds
- Boss battle music
- Volume control (↑/↓ arrows)
- Sound toggle (S key)

## 🎮 Controls

| Key | Function |
|-----|----------|
| ← → or A/D | Move ship |
| SPACE | Shoot lasers |
| S | Toggle sound effects |
| ↑ | Increase volume |
| ↓ | Decrease volume |
| L | Load saved game |
| Ctrl+S | Save current game |
| H | View high scores |
| ESC | Return to menu/exit |
